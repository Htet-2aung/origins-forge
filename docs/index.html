The Origins Forge.

A proprietary Internal Developer Platform (IDP) designed to accelerate project delivery by 400%. Built for engineers, by engineers.

Installation Protocol

Run the following in your terminal to initialize the Forge environment:

# Clone the IDP core
git clone https://github.com/Hsu212/origins-cli.git

# Link the global command
cd origins-cli && pip install --editable .

# Sync Blueprint manifest
origins sync

CLI Capability Matrix

origins clone [ID]

Pulls the master template from GitHub, scrubs the git history, and initializes a new client workspace in /projects.

Example: origins clone web-mvp

origins setup

The "One-Command" installer. Auto-detects project stack, creates virtual environments, installs dependencies, and boots localhost.

Context: Run inside project folder

origins ask [MSG]

Terminal-integrated AI trained on Origins engineering standards. Use it for debugging, refactoring, or generating logic.

Cost: $0.00 (Company Key)

origins ship & deploy

Wraps the Vercel/Render API. Ships current branch to production and provides a live URL for client review instantly.

Output: Production URL

Troubleshooting

Port already in use (EADDRINUSE)
Run origins kill [PORT] to instantly terminate the process blocking your development server.
Blueprints not appearing after sync
Check your internet connection and ensure your OpenAI/GitHub tokens are set in ~/.origins/config.json.
