The Origins Forge.

An intelligent Python-powered toolchain for automated app generation, dependency orchestration, and global deployment.

Explore Build Engine

Installation

# Clone the forge
git clone https://github.com/Htet-2aung/origins-forge.git
# Install globally in editable mode
cd origins-forge && pip install -e .
# Sync blueprints
origins sync

One-Click Setup

The bootstrap command is designed for new engineers. It automatically detects your OS and installs Git, Node.js, Python 3.12, and Docker.

origins bootstrap Supports MacOS (Brew) | Windows (Winget) | Linux (APT)

AI Factory: Build Engine

Wizard Mode

Interactive prompt to select your framework (FastAPI, Next.js, Flask), Database (Postgres, MongoDB, SQLite), and deployment features.

origins build --wizard

Swarm Mode

Deploy parallel AI agents using a ThreadPoolExecutor to architect core API, dependencies, and documentation simultaneously.

origins build --swarm

Universal Get

Stop remembering OS-specific package managers. origins get handles the registry for you.

origins get node

origins get python

origins get docker

origins get git

Smart Stack Setup

Auto-Stack Detection

Forge scans for package.json or requirements.txt and handles the environment initialization:

  • Virtual Env (venv) creation for Python
  • OS-aware pathing for pip binaries
  • Automated dev-server startup (Uvicorn / NPM)
Live Output 
> origins setup
Detected Python/FastAPI Project...
Creating virtual environment...
Installing requirements.txt...
Starting server at http://127.0.0.1:8000

Cloud Operations

origins ship

Atomic commit and push to GitHub Main.

origins ship -m "v1.1"

origins deploy

One-click push to Vercel or Railway.

origins deploy

origins secret

Generate cryptographically secure keys.

origins secret --length 64

System Health

origins update

Self-repairing update engine. Pulls latest source code, bootstraps build tools (setuptools/wheel), and re-links the CLI binary.

origins update

origins test-api

Stress tests the Gemini 1.5 Flash connectivity and GitHub API authentication. Measures real-world AI engine latency.

origins test-api
