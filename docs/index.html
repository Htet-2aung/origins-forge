The engineering backbone of Origins. This toolset automates blueprint generation, AI-assisted debugging, and multi-modal cloud deployments.

01. Installation

Ensure your SSH keys are added to GitHub before cloning blueprints.

$ git clone https://github.com/Hsu212/origins-cli.git
$ cd origins-cli
$ pip install --editable .
$ origins sync

02. Command Protocol

origins clone

Spawns a client workspace into /projects.

origins setup

Automatic stack detection & localhost launch.

origins ask

Interactive terminal session with Origins AI.

origins ship

Aggregated Git workflow for rapid pushes.
