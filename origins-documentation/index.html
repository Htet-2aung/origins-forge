Getting Started

To install the forge toolset globally on your local machine, run:

git clone https://github.com/Hsu212/origins-cli.git
cd origins-cli
pip install --editable .
origins sync

Command Reference

origins clone [id]

Clones a master blueprint into origins-cli/projects/. Detaches git history automatically.

origins setup

Detects project type, installs node_modules or venv, and starts local server.

origins ask "question"

Queries the Origins AI assistant for coding help directly in the terminal.

origins ship -m "msg"

Automated Add-Commit-Push workflow for rapid versioning.
